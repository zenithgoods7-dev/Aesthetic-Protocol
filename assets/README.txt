DROP YOUR REAL EBOOK COVERS HERE
================================

The site shows a built-in styled placeholder cover for each book until you
add your real cover image. To use your actual covers, save them in THIS
folder with these exact filenames:

    aesthetic-dominion.jpg     ->  Book I  (The Aesthetic Dominion)
    cardinal-mind.jpg          ->  Book II (The Cardinal Mind)
    wealth-architecture.jpg    ->  Book III (The Wealth Architecture)

Tips
----
- JPG or PNG both work (keep the .jpg name, or change it in index.html).
- Recommended ratio 3:4 (portrait), e.g. 900 x 1200 px.
- The moment a file with the right name exists, the site uses it
  automatically and hides the placeholder — no code changes needed.
